# study-tracker

A full-stack web application designed to help users enhance their productivity through focused study sessions using the Pomodoro Technique. It allows users to register, log in, manage study tasks, utilize a configurable Pomodoro timer, and track their study history.

## Detailed Usage Instructions

For detailed instructions on how to set up, run, and use this application, please refer to:

- [English Instructions (README.en.md)](./README.en.md)
- [日本語の取扱説明書 (README.ja.md)](./README.ja.md)

## Key Technologies

*   **Backend:** Bun, Elysia.js
*   **Frontend:** React (with Vite), TypeScript
*   **Styling:** Tailwind CSS
*   **Database:** SQLite
*   **Testing:** `bun:test` (backend), Vitest & React Testing Library (frontend)
